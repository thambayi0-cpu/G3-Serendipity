Detailed Description

Function declarations for the bookstore inventory menu system.

These functions operate on a vector of bookType objects and use the static field bookType::bookCount to track the total number of books in the inventory. The maximum number of books allowed is 20.

◆ addBook()

void addBook ( std::vector< bookType > & books)

Add a new book to the inventory (maximum of 20 books).

Collects book information from the user and appends a new entry to the vector. The addition is only allowed if bookType::bookCount < 20.

Parameters
booksReference to the inventory vector.
Precondition
bookType::bookCount < 20. If bookCount == 20, the function not run.
Postcondition
  • If bookType::bookCount < 20, a new book is added and bookCount is incremented.
  • If bookType::bookCount == 20, no change is made.

◆ deleteBook()

void deleteBook ( std::vector< bookType > & books)

Delete a book from the inventory.

Uses lookUpBook() to locate a book by title or ISBN, and removes it if found.

Parameters
booksReference to the inventory vector.
Precondition
bookType::bookCount > 0 (cannot delete from an empty inventory).
Postcondition
  • If the book exists, it is erased from the vector and bookType::bookCount is decremented.
  • If not found, no modification is made.

◆ editBook()

void editBook ( std::vector< bookType > & books)

Edit an existing book in the inventory.

Uses lookUpBook() to locate a book by title or ISBN, then allows modifying it.

Parameters
booksReference to the inventory vector.
Precondition

Postcondition
