\doxysubsection{Detailed Description}
Function declarations for the bookstore inventory menu system. 

These functions operate on a vector of \doxylink{classbookType}{book\+Type} objects and use the static field \doxylink{classbookType_a1e98b5e92ac7ffaba2123bfce987f8c6}{book\+Type\+::book\+Count} to track the total number of books in the inventory. The maximum number of books allowed is 20.
