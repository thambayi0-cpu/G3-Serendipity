\doxysubsection{Detailed Description}
Declaration of the \doxylink{classbookType}{book\+Type} class for managing book inventory data. 

This class encapsulates bibliographic information, inventory quantity, and pricing (wholesale/retail). Quantity is capped at 100, and pricing fields are capped at 999.\+99. A static counter tracks the number of \doxylink{classbookType}{book\+Type} objects created.
