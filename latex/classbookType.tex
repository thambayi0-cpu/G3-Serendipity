\doxysubsection{Detailed Description}
Stores metadata and inventory information for a book. 

Fields\+:
\begin{DoxyItemize}
\item title\+: Book title
\item isbn\+: ISBN identifier
\item author\+: Author name
\item publisher\+: Publisher name
\item date\+: Date string (mm/dd/yy)
\item qty\+: Quantity in stock (capped at 100)
\item wholesale\+: Wholesale price (capped at 999.\+99)
\item retail\+: Retail price (capped at 999.\+99)
\end{DoxyItemize}

A static field tracks the number of \doxylink{classbookType}{book\+Type} objects created.
